Question:-

How to use “InterProcessSignalPropagator” in your Qt applications?

 

Answer:-

To use “InterProcessSignalPropagator” in Qt applications following things should be done

1.      Considering your applications, decide signals to be propagated to other process.

2.      Derive a class having all above said signals from “QInterProcessSignalPropogator” (defined in “InterProcessSignalPropagator/InterProcessSignalPropagatorLibrary/qinterprocesssignalpropagator.h”)

3.      Link your application with “InterProcessSignalPropagator’ library

4.      Create an instance of this said class in both the application within which you want to propagate the signal

1.      Out of which the first staring application should have the “server” component of “InterProcessSignalPropogator”

2.      And the later should have the “client” component of “InterProcessSignalPropogator”

5.      Connect the “outgoing signals” to the signals of said derived class using “Queued Connection”

6.      Connect the “incoming signals” of said derived class to appropriate slots from your application using “Queued Connection”
